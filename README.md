# Apsol — Landing de Servicios

Esta es la landing page oficial de **Apsol** para la digitalización de operaciones y consultoría de procesos para PyMEs y empresas en general.

## Características
- **Diseño Premium**: Interfaz moderna y minimalista con soporte nativo de modo oscuro y claro.
- **Micro-animaciones**: Transiciones de scroll suaves y contadores de estadísticas dinámicos.
- **Carruseles Táctiles**: Carruseles interactivos de logotipos de clientes y testimonios optimizados con `requestAnimationFrame` para máximo rendimiento y soporte táctil inercial.
- **Agendamiento Directo**: Integración asíncrona de widget de Calendly para agendar llamadas de consulta gratuita de manera inmediata.

## Tecnologías Utilizadas
- HTML5 (Estructura semántica, accesibilidad)
- CSS3 Vanilla (Estilos fluidos)
- JavaScript nativo (Interactividad sin dependencias externas pesadas)

## Desarrollo Local
Para levantar el servidor local de prueba, podés ejecutar:
```bash
python -m http.server 8000
```
Y acceder a: [http://localhost:8000](http://localhost:8000)
